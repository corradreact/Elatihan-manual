# Panduan Pendaftaran Pengguna

::: info Tujuan
Membantu pengguna baharu membuat pendaftaran ke dalam Sistem E-Latihan menggunakan NRIC. Setelah pendaftaran berjaya, pengguna boleh log masuk menggunakan kata laluan apisim.
:::

## 1. Langkah-langkah Pendaftaran Pengguna
::: tip Langkah 1
1. Akses Halaman Utama
2. Pada halaman utama, pilih menu **Daftar Pengguna**
:::

## 2. Masukkan NRIC
::: tip Langkah 2
1. Isikan NRIC pengguna pada ruangan yang disediakan
2. Pastikan NRIC dimasukkan tanpa simbol dan dalam format 12 digit
3. Klik Daftar untuk menghantar permohonan pendaftaran

![daftar](/image/kakitangan/pendaftaran-p.png)

:::

## 3. Pengesahan Pendaftaran
::: tip Nota
1) Setelah menekan butang Daftar:
![Button daftar](/image/kakitangan/btn-dftr.png)
- Sistem akan menyemak NRIC
- Jika NRIC sah dan belum pernah didaftarkan, sistem akan mencipta akaun pengguna baharu
2) Sistem akan memaparkan mesej seperti berikut:
**"Pendaftaran berjaya. Sila log masuk menggunakan kata laluan apisim."**
