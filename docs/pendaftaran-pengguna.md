# Panduan Pendaftaran Pengguna

## Tujuan
Membantu pengguna untuk mendaftar akaun baru dalam sistem MyReCourt.

---

## Langkah-langkah Pendaftaran

### 1. Akses Sistem MyReCourt

::: info Langkah 1
1. Buka pelayar web anda (contohnya Google Chrome).  
2. Layari URL: https://myrecourt.kedah.gov.my/  
3. Anda akan melihat paparan log masuk sistem.
:::

### 2. Daftar Akaun Baru

1. Klik pautan **"Daftar Akaun Baru"** pada skrin log masuk.  
2. Isikan maklumat berikut:
   - Nama Penuh  
   - Nombor Kad Pengenalan  
   - Emel  
   - Kata Laluan  
3. Tandakan kotak persetujuan terma & syarat.  
4. Klik butang **Daftar**.

::: tip Nota
Pastikan emel yang dimasukkan adalah sah kerana pengesahan akaun akan dihantar ke emel tersebut.
:::
