# Panduan Penilaian Latihan
::: info Tujuan
Peserta latihan dapat membuat penilaian untuk latihan yang telah dihadiri.
:::
## Membuat Penilaian

### 1. Klik "Buat Penilaian"
::: tip Langkah 1
1. Log masuk ke sistem
2) Pilih menu **Penilaian**
3. Klik butang **"Buat Penilaian"**

:::
---


### 2. Semua 4 bahagian **WAJIB** diisi sebelum dihantar.
### Bahagian A: Pelaksanaan Kursus

![Bahagian A](/image/kakitangan/bahagiana.png)

### Bahagian B: Perkhidmatan dan Kemudahan Kursus/latihan (Jika Berkenaan)

![Bahagian B](/image/kakitangan/bahagianb.png)


### Bahagian C: Penilaian Penceramah/Konsultan
![Bahagian C](/image/kakitangan/bahagianc.png)

### Bahagian D: Rumusan dan Cadangan Penambahbaikan

![Bahagian D](/image/kakitangan/bahagiand.png)

---

## 3. Menyerahkan Penilaian

::: tip Langkah 3
Setelah mengisi semua 4 bahagian:
1. Periksa semua jawapan
2. Klik **"Hantar"**
:::

### 4. Popup Pengesahan

::: tip Langkah 4
1. Popup akan muncul untuk pengesahan akhir:
**"Hantar**
![Sahkan](/image/kakitangan/Sahkan.png)

Pilih:
- **Batal** - Kembali ke borang untuk ubah jawapan
- **Hantar** - Sahkan dan hantar penilaian
Selepas klik **"Hantar"**, penilaian akan dihantar.
![Sahlan](/image/kakitangan/success4.png)
:::

---
::: warning Nota
- Semua 4 bahagian **WAJIB** diisi
- Tidak boleh ada soalan yang kosong
- Tidak dapat diubah setelah dihantar
- Isi segera setelah kursus selesai
:::
