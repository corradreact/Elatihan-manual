# Panduan Pengurusan Penceramah (Penilaian Dalaman)

::: info Tujuan
Membantu admin mengurus senarai penceramah bagi sesuatu latihan. Panduan ini menerangkan cara memilih latihan, melihat senarai penceramah, menambah penceramah baharu, mengedit maklumat penceramah dan memadam penceramah dari sistem.
:::

## Senarai Latihan
### 1. Akses Menu Laporan Latihan

::: tip Langkah 1
1) Log masuk ke dalam sistem
2) Pilih menu **Laporan Latihan**
3) Sistem akan memaparkan senarai latihan beserta tajuk, lokasi dan tarikh
:::
![Laporan Latihan](/image/admin/laporan-latihan.png)

## 2. Pilih Tindakan Penceramah
Setiap latihan mempunyai beberapa tindakan termasuk Peserta, Penceramah, dan Penilaian.

::: warning Tindakan
Klik butang **"Penceramah"** untuk mengurus senarai penceramah bagi latihan tersebut.
![Peceramah](/image/admin/penceramah.png)
:::

### Pengurusan Penceramah
::: tip Langkah 2
1) Butang ini akan membawa admin ke halaman **Penceramah (Penilaian Dalaman)**.
2) Admin boleh:
- Melihat senarai penceramah sedia ada
- Menambah penceramah baharu
- Mengedit nama penceramah
- Memadam penceramah
:::
## 3. Menambah Penceramah Baharu
::: tip Langkah 3
1) Isikan nama penceramah pada ruangan Nama Penceramah.
2) Klik butang **Tambah**.
![Tambah](/image/admin/p-tambah.png)

3) Sistem akan memaparkan mesej berjaya seperti:
**"Penceramah berjaya ditambah ke pangkalan data"**
![Success2](/image/admin/success2-p.png)

4) Nama penceramah akan terus muncul dalam senarai.
![Senarai Penceramah](/image/admin/senarai-penceramah.png)
:::

## 4. Mengemaskini Nama Penceramah
Untuk mengubah maklumat penceramah:
::: tip Langkah 4
1) Klik butang **Edit** di sebelah nama penceramah
![Edit Pe](/image/admin/edit-pe.png)
2) Ruangan input akan aktif untuk dikemaskini
3) Masukkan nama baharu, klik **Simpan**
![Simpan](/image/admin/btn-simpan.png)

4) Sistem memaparkan mesej:
"Penceramah berjaya dikemaskini"
![Simpan](/image/admin/success3-p.png)
:::
## 5. Memadam Penceramah
Jika penceramah tidak lagi terlibat dalam latihan:

::: tip Langkah 5
1. Klik butang **Padam** di sebelah nama penceramah
![Padam](/image/admin/padam.png)
2. Sistem memaparkan mesej:
"Penceramah dipadam dari pangkalan data"
![Simpan](/image/admin/success4-p.png)
:::

::: warning Nota
- Pastikan nama penceramah adalah tepat sebelum disimpan kerana ia akan dipaparkan pada borang penilaian latihan dalaman.

:::
