# Panduan Pendaftaran Latihan oleh Admin Jabatan

::: info Tujuan
Panduan ini bertujuan membantu Admin Jabatan mendaftarkan latihan setelah permohonan telah diluluskan oleh Yang Dipertua (YDP) melalui Sistem Pengurusan Latihan Online (Smart Training).
:::

## Langkah-langkah Pendaftaran Latihan
### 1. Log Masuk sebagai Admin Jabatan

::: tip Langkah 1

1. Log masuk ke dalam Sistem Pengurusan Latihan Online (Smart Training) menggunakan peranan **Admin Jabatan**
2. Pilih menu Dashboard Admin
3. Sistem akan memaparkan Dashboard (Tindakan) yang memfokuskan kepada tindakan yang perlu dibuat oleh **Admin Jabatan**

![Dashboard](/image/dashboard/dashboard-aj.png)
:::

### 2. Mengenal Pasti Permohonan yang Diluluskan

::: tip Langkah 2

1) Pada senarai tindakan, kenal pasti permohonan yang:
- Berstatus **Daftar**
2) Permohonan ini telah melalui kelulusan di Bahagian F: Peringkat Kelulusan
3) Klik butang **Daftar** pada lajur Tindakan
![Daftar](/image/dashboard/daftar.png)

:::

### 3. Menyemak Kelulusan YDP
::: tip Langkah 3
1. Sistem akan memaparkan butiran permohonan yang telah diluluskan
2. Semak maklumat berikut di **Bahagian F: Peringkat Kelulusan:**
- Status YDP: Diluluskan
- Ulasan YDP
:::

### 4. Mengisi Maklumat Pendaftaran Latihan

::: tip Langkah 4

1) Pada skrin pendaftaran latihan, isikan maklumat berikut:
- Course ID (Wajib)
- Pastikan Course ID adalah tepat dan sah mengikut rekod penganjur latihan
:::
![Bukti](/image/dashboard/ulasan-aj.png)

### 5. Mengesahkan dan Mendaftar Latihan

::: tip Langkah 5 

1. Setelah semakan selesai, klik butang **Daftar**
![Hantar Ulasan](/image/dashboard/daftar.png)
2. Sistem memaparkan mesej:
"Anda pasti mahu didaftarkan permohonan ini ke dalam rekod (status = Diluluskan)"
![Sahkan](/image/dashboard/sahkan-daftar.png)
3. Klik **Batal** sekiranya ingin membuat semakan semula
![Batal](/image/dashboard/batal.png)

4. Sistem akan mengemas kini status permohonan secara automatik.
:::
::: warning Nota
- Pendaftaran latihan hanya boleh dibuat selepas kelulusan YDP
- Course ID adalah maklumat **WAJIB** untuk tujuan rekod pengesanan latihan dan peserta latihan
- Pastikan maklumat yang didaftarkan adalah tepat bagi mengelakkan sebarang isu pentadbiran
- Kegagalan untuk mendaftar Course ID yang tepat akan menyebabkan peserta tidak dapat melihat Latihan yang perlu disertai.
:::
