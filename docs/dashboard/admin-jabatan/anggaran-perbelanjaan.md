# Anggaran Perbelanjaan oleh Admin Jabatan

::: info Tujuan 
Panduan ini bertujuan membantu Admin Jabatan membuat semakan dan pengisian anggaran perbelanjaan latihan setelah permohonan diluluskan oleh Pengarah Jabatan, sebelum dihantar untuk proses seterusnya.
:::

## 1. Langkah-langkah Mengisi Anggaran Perbelanjaan
::: tip Langkah 1
1. Log masuk ke dalam Sistem E-Latihan sebagai Admin Jabatan
2. Pilih menu Dashboard Admin
3. Senarai permohonan yang telah mendapat Sokongan Pengarah Jabatan akan dipaparkan

![Bahagian E](/image/dashboard/bahagian-e.png)
:::

## 2. Memilih Permohonan untuk Semakan
::: tip Langkah 2
1. Pada senarai permohonan, kenal pasti permohonan yang berstatus:
- Bahagian E
2. Klik butang "**Isi Bahagian E**" pada lajur Tindakan
![Isi Bahagian E](/image/dashboard/isi.png)
:::

## 3. Menyemak Sokongan Pengarah Jabatan

::: tip Langkah 3
1. Setelah borang dibuka, skrin akan memaparkan maklumat permohonan lengkap
2. Admin Jabatan perlu menyemak:
- Bahagian D: Ulasan / Sokongan Pengarah Jabatan
- Status sokongan (Diluluskan)
- Ulasan yang diberikan oleh Pengarah Jabatan
- Pastikan permohonan telah diluluskan sebelum meneruskan pengisian anggaran perbelanjaan
:::
![Bahagian D](/image/dashboard/bahagian-d.png)

## 4. Mengisi Anggaran Perbelanjaan (Bahagian E)

::: tip Langkah 4
1. Di Bahagian E: Semakan Permohonan
2. Isikan maklumat berikut berdasarkan dokumen dan kelulusan berkaitan:

- Tarikh Terima Borang Permohonan
- Tarikh Latihan (Mula)
- Yuran Pendaftaran (RM)
- Kos Penginapan (jika berkaitan) (RM)
- Tuntutan Perjalanan / Tiket Penerbangan (jika berkaitan) (RM)
- Lain-lain Kos (jika berkaitan) (RM)

3. Sistem akan mengira Jumlah (RM) secara automatik
:::
![Semakan Permohonan](/image/dashboard/semakan-permohonan.png)

## 5. Menghantar Anggaran Perbelanjaan
::: tip Langkah 5
1. Pastikan semua maklumat perbelanjaan telah diisi dengan betul
2. Semak semula jumlah anggaran yang dipaparkan
3. Klik butang **Hantar** untuk menyimpan dan menghantar semakan
![Hantar](/image/dashboard/hnatar.png)
4. Klik **Batal** sekiranya ingin membatalkan atau membuat semakan semula
![Batal](/image/dashboard/batal.png)
:::

::: warning Nota
- Pengisian anggaran perbelanjaan hanya boleh dibuat selepas sokongan Pengarah Jabatan
- Nilai kos hendaklah berdasarkan dokumen sokongan atau kadar yang ditetapkan
- Kesilapan pengisian boleh menjejaskan proses kelulusan seterusnya
:::
