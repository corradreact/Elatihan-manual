# Panduan Tambah Pengguna

::: info Tujuan
Membantu admin menambah pengguna baharu ke dalam sistem melalui modul Pengurusan Pengguna.
:::

## Langkah-langkah Tambah Pengguna
### 1. Akses Menu Pengurusan Pengguna

::: tip Langkah 1

1) Log masuk ke dalam sistem
2) Pilih menu **Tambah Pengguna** (Admin Jabatan Sahaja) 
3) Sistem akan memaparkan senarai pengguna serta butang tindakan
:::
![Senarai](/image/admin/senarai-pengguna.png)

### 2. Tambah Pengguna
::: tip Langkah 2
1) Klik butang **Tambah Pengguna** untuk membuka borang pendaftaran pengguna baharu.
![Tambah Pengguna](/image/admin/tambah-pengguna.png)
2) Lengkapkan maklumat pengguna seperti berikut:
![Tambah Pengguna](/image/admin/tmbh-pengguna.png)
3) Semak semua maklumat yang dimasukkan
4) Klik butang Simpan
![Simpan](/image/admin/btn-simpan.png)
5) Sistem akan memaparkan mesej "Pengguna berjaya ditambah" jika proses selesai
:::

::: warning Nota
- Pastikan emel pengguna adalah sah dan aktif.
- Peranan (Role) menentukan akses dalam sistem.
- Nama Pengguna mungkin tidak boleh ditukar selepas didaftarkan bergantung pada konfigurasi sistem.
:::
