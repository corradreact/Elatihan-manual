# Panduan Blast Emel Kursus

::: info Tujuan
Membantu admin menghantar emel kursus secara pukal (blast) kepada pemohon berdasarkan tarikh kursus dan jenis kursus yang dipilih.
:::

## Langkah-langkah Blast Emel Kursus
### 1. Akses Modul Blast Emel

::: tip Langkah 1

1. Log masuk ke dalam sistem

2. Pilih menu **Blast Emel Kursus**

3. Sistem akan memaparka senarai kursus, jenis template, protonton dan serta senarai pemohon 
:::

### 2. Pilih Tarikh & Jenis Kursus
::: tip Langkah 2
1. Pilih Tarikh & Jenis Kursus

2. Admin perlu memilih tarikh kursus dan jenis kursus untuk mendapatkan senarai pemohon yang berkaitan.

3. Sistem akan memaparkan senarai pemohon mengikut kursus yang dipilih.

4.Admin boleh:
- Menyemak alamat emel penerima
- Mengubah emel jika terdapat maklumat tidak tepat
- Menambah penerima tambahan jika perlu

:::

::: warning Nota
- Jika jumlah penerima lebih daripada 30 orang, admin perlu mengisi sendiri alamat emel penerima pada ruangan yang disediakan.
- Admin juga boleh mengubah alamat emel bagi penerima sebelum menghantar emel.
:::

### 3. Template Emel
::: tip Langkah 3 

1) Admin boleh memilih template emel yang disediakan dalam sistem.

Template boleh:
- Dipilih
- Dibaca isi kandungannya
- Dikemaskini atau diubah suai jika perlu
:::

### 4. Tambah CC & Lampiran
::: tip Langkah 4
1) Admin boleh memasukkan penerima tambahan melalui CC serta menambah dokumen lampiran.

2) Maklumat yang boleh diisi termasuk CC (jika diperlukan) serta lampiran dalam format PDF, DOC, DOCX, JPG atau PNG.

:::
::: warning Nota
- Saiz maksimum fail: 5MB
:::


### 5. Menghantar Blast Emel
::: tip Langkah 5

1) Setelah semua maklumat lengkap, admin boleh menghantar emel.

2) Semak semua maklumat penerima.

3) Semak atau kemas kini template emel.

4) Masukkan CC dan lampiran yang diperlukan.
:::

#### Butang "Menghantar Emel"
::: tip Menghantar Emel 
- Klik butang biru **"Menghantar emel"** last Emel untuk menghantar
- Jika terdapat medan yang belum lengkap, sistem akan memaparkan mesej ralat.
- Jika berjaya, sistem akan memaparkan mesej pengesahan.
- Sistem akan memaparkan mesej "Emel berjaya dihantar kepada penerima".
- Emel dihantar kepada semua penerima dalam senarai.
::
