# Panduan Menambah Template Emel

::: info Tujuan
Membantu admin menambah template emel baharu yang akan digunakan untuk jemputan atau hebahan latihan dalaman atau luaran kepada peserta.
:::


## Langkah-langkah Menambah Template Emel
### 1. Akses Menu Template Emel

::: tip Langkah 1

1) Log masuk ke dalam sistem
2) Pilih menu **Tetapan**
3) Klik pada kad Template Emel untuk membuka borang tetapan
:::
![Menu](/image/admin/menu-template.png)

### 2. Isikan Maklumat Template Emel

::: tip Langkah 2
Isikan maklumat berikut pada borang Template Emel:

- Nama Template
- Subjek Emel
- Kandungan Emel (boleh dirangka menggunakan Panduan HTML)
- Sistem juga memaparkan ruang Pratonton (HTML) untuk melihat hasil format HTML secara langsung.
:::


![Template](/image/admin/template-emel.png)
### 3. Simpan Template Emel Baharu

::: tip Langkah 3

1) Semak semua maklumat yang telah diisi
2) Klik butang **Simpan** untuk menambah template emel baharu
![Simpan](/image/admin/btn-simpan.png)
3) Sistem akan memaparkan mesej berjaya setelah template disimpan
:::

### 4. Edit Template Emel Sedia Ada
::: tip Langkah 4
1) Admin boleh mengemaskini template emel yang telah disimpan. Pada senarai template, klik butang **Edit**.

![Edit](/image/admin/edit.png)  

2) Sistem akan memaparkan maklumat sedia ada dalam borang
3) Kemas kini nama, subjek atau kandungan template
4) Klik butang **Simpan** untuk menyimpan perubahan
![Simpan](/image/admin/btn-simpan.png)
:::

### 5. Padam Template Emel Sedia Ada

::: tip Langkah 5

1) Klik butang **Padam** pada template yang ingin dipadam
![Padam](/image/admin/padam.png)
2) Sistem memaparkan  template berjaya dipadam
![Padam](/image/admin/success1-padam.png)
3) Template akan dipadam daripada senarai
:::


::: warning Nota
- Pastikan semua maklumat wajib diisi sebelum menekan Simpan.
- Kandungan HTML perlu dirangka dengan betul untuk mengelakkan paparan emel menjadi tidak teratur.
- Template yang disimpan boleh digunakan semasa penghantaran blast emel.
:::
