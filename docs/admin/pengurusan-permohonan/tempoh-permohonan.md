# Panduan Tetapan Tempoh Akhir Permohonan Luaran


::: info Tujuan
Membantu admin menetapkan tempoh akhir (cutoff) bagi permohonan latihan luaran. Tetapan ini menentukan bilangan hari sebelum tarikh latihan yang masih dibenarkan untuk pengguna menghantar permohonan.
:::

## 1.  Langkah-langkah Mengemaskini Tempoh Akhir Permohonan.
::: tip Langkah 1
1. Log masuk ke dalam sistem
2. Pilih menu **Tetapan**
3. Klik pada kad **Permohonan Luaran** untuk membuka borang tetapan cutoff hari
:::
![Laporan Latihan](/image/admin/tetapan.png)

## 2. Masukkan Tempoh Akhir (Hari)
::: tip Langkah 2
1. Isikan bilangan hari yang menentukan tempoh akhir permohonan sebelum tarikh latihan.
2. Contoh:
- Jika diisi 7, pengguna hanya boleh menghantar permohonan sehingga 7 hari sebelum tarikh latihan.
- Jika diisi 2, permohonan dibenarkan sehingga 2 hari sebelum latihan bermula.
![Tempoh akhir](/image/admin/tempoh-akhir.png)

3. Semak bilangan hari yang telah dimasukkan. Klik butang **Simpan**
![Simpan](/image/admin/btn-simpan.png)
4. Sistem akan memaparkan mesej: "Tetapan berjaya dikemaskini".
![Success](/image/admin/success5.png)

:::


::: warning Nota
- Tetapan ini memberi kesan kepada semua pengguna yang ingin membuat permohonan latihan luaran.
- Pastikan bilangan hari ditetapkan mengikut polisi organisasi.
- Perubahan berkuat kuasa serta-merta selepas disimpan.
:::
